Nashville Housing Data Cleaning in SQL

This project cleaned raw housing data and transformed it into an SQL Server to make it more usable for analysis. In this project, COALESCE was used to clean data by filling in missing property addresses and ensuring data consistency. Furthermore, I extracted individual components (address, city, state) using POSITION and SPLIT_PART. Converted categorical fields (Y/N) into readable values (Yes/No) for clarity in analysis. Applied Common Table Expressions (CTEs) and subqueries to identify and eliminate redundant records. Optimized data by dropping unnecessary columns and structured the dataset for efficient querying. Developed an interactive Tableau dashboard to present insights on property trends, ownership patterns, and sales history.
