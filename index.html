Nashville Housing Data Cleaning in SQL

This project cleaned raw housing data and transformed it in a SQL Server to make it more usable for analysis. In this project COALESCE was used to clean data by filling missing property addresses and ensure data consistency. Furthermore, I extracted individual components (address, city, state) using POSITION and SPLIT_PART. Converted categorical fields (Y/N) into readable values (Yes/No) for clarity in analysis. Applied Common Table Expressions (CTEs) and subqueries to identify and eliminate redundant records. Optomized data by dropping unnecessary columns and structured the dataset for efficient querying. Developed an interactive Tableau dashboard to present insights on property trends, ownership patterns, and sales history.
